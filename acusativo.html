Guía visual del caso acusativo en ruso para hispanohablantes

¿Para qué sirve el acusativo?
Es el caso del objeto directo — la persona o cosa que recibe la acción del verbo.
Я вижу книгу → Veo el libro · Я люблю маму → Quiero a mamá
Sustantivos — terminaciones en acusativo
Masculino Femenino Neutro Plural
Masculino
Animado (persona/animal)
студент → студент-а
кот → кот-а
Inanimado (cosa)
стол → стол (= nominativo)
город → город
Femenino
-а → -у
книга → книг-у
мама → мам-у
-я → -ю
земля → земл-ю
семья → семь-ю
Neutro
Sin cambio (= nominativo)
окно → окно
море → море
имя → имя
Plural
Animado = genitivo pl.
студенты → студент-ов
Inanimado = nominativo pl.
книги → книги
столы → столы

Truco mnemotécnico: Las terminaciones del acusativo femenino -у / -ю suenan como "u" de "único objeto" — exactamente lo que marca el acusativo: el objeto único de la acción.
