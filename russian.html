Зачем нужен винительный падеж?
Это падеж прямого дополнения — лица или предмета, на которое направлено действие глагола.
Я вижу книгу · Я люблю маму
Существительные — окончания в винительном падеже
Мужской Женский Средний Множ. число
Мужской
Одушевлённые (люди/животные)
студент → студент-а
кот → кот-а
Неодушевлённые (предметы)
стол → стол (= именит.)
город → город
Женский
-а → -у
книга → книг-у
мама → мам-у
-я → -ю
земля → земл-ю
семья → семь-ю
Средний
Без изменений (= именит.)
окно → окно
море → море
имя → имя
Множ. число
Одушевл. = родит. мн.ч.
студенты → студент-ов
Неодушевл. = именит. мн.ч.
книги → книги
столы → столы

нов-ый стол

нов-ую книгу
Средний
-ое / -ее (= имен.)
нов-ое окно

любитьлюбить

Подсказка: Окончания женского рода в винительном падеже -у / -ю — самые простые: они всегда меняются и легко узнаются.

Общая информация

  • Алфавит: 33 буквы (кириллица)
  • Грамматика: 6 падежей, 3 рода, глагольные виды
  • Носители: более 250 миллионов человек
  • Пятый по распространённости язык в Интернете
  • Официальный язык: Россия, Беларусь, Казахстан, Кыргызстан
